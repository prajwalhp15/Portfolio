# Personal Portfolio - Prajwal HP

Welcome to my personal portfolio! This website showcases my projects, skills, certifications, and professional experience.

## 🌐 Live Demo
[View Live Portfolio](https://portfolio-np7i.onrender.com/)

## 🛠 Technology Stack
- **Frontend:** React, Vite, TypeScript
- **Styling:** Tailwind CSS, shadcn/ui
- **Build Tool:** Vite
- **Deployment:** Render

## 📂 Project Structure
├─ public/ # Public assets like images, favicon
├─ src/ # Source code
│ ├─ components/ # Reusable React components
│ ├─ pages/ # Portfolio pages (About, Projects, Contact)
│ └─ App.tsx # Main app entry
├─ dist/ # Production build output
├─ package.json
├─ vite.config.ts
└─ README.md

## 🚀 Features
- Clean, modern design
- Responsive layout (desktop & mobile)
- Projects section with descriptions and live links
- Skills & tools showcase
- Contact form (or email link)
- Smooth animations using CSS & Tailwind

## ⚡ Installation (for local development)
1. Clone the repo:
```bash
git clone https://github.com/prajwalhp15/code-prism-ai.git
cd code-prism-ai
npm install
npm run dev
npm install -g serve
serve -s dist
```
📌 Deployment

The portfolio is deployed on Render: Live Link

Ensure vite.config.ts has base: "./" for correct asset paths.
