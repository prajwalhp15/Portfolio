## 🎨 Design Philosophy & UX System

* **Visual Theme**: Dark mode first (`#0A0A0A` background) drawing inspiration from premium tech brands like Vercel, Linear, and Stripe.
* **Typography**: Clean, geometric sans-serif headings using Google Font `Plus Jakarta Sans` paired with `Inter` for body copy.
* **Micro-Animations**: Subtle page reveals, blur entries, and layout transitions handled smoothly via `framer-motion` without performance clutter.
